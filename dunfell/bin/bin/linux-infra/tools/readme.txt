Tools for Linux develoment will be in this directory.

Recomendation:
 Clone this repo and put it in your path.  Such as:

 $ mkdir ~/bin
 $ cd ~/bin
 $ git clone gitolite@at-git:linux-infra.git
 
Add a line like this to your ~/.bashrc
 PATH="$PATH:~/bin/linux-infra/tools"

Whenever something gets added to the tools, you can update by doing:
 $ cd ~/bin/linux-infra
 $ gg
